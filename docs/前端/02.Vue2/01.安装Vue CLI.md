# 安装Vue CLI

```shell
npm install -g @vue/cli
```

查看版本

```shell
vue --version
```





## 创建Vue CLI 项目

进入到要放在项目的文件夹

```shell
vue create 项目名
```



## 启动Vue CLI项目

进入项目文件夹

```shell
npm run serve
```



## 安装 Element

**用cmd打开项目文件夹中安装**

```shell
npm i element-ui -S
```
