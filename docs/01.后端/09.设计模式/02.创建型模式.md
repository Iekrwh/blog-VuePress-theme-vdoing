# 创建型模式

- 创建型模式关注点“**怎样创建出对象？**”

- “**将对象的创建与使用分离**”。

- **降低系统的耦合度**

- **使用者无需关注对象的创建细节**

  - 对象的创建**由相关的工厂来完成**；（各种工厂模式）

  - 对象的创建**由一个建造者来完成**；（建造者模式）

  - 对象的创建由原来对象**克隆**完成；（原型模式）

  - 对象始终在系统中**只有一个实例**；（单例模式）
