# 标识符

标识符：给类，方法，变量起名字的符号

由数字、字母、下划线、美元符($)组成的

不能以数字开头，不能为关键字，区分大小写

小驼峰命名法(针对方法、变量命名)

1.标识符是一个单词时，首字母小写   name

2.由多个单词组成时，第一个单词首字母小写，其他的首字母大写

firstName

大驼峰命名法(针对类)

1.首字母大 Student

2.每个单词首字母大写  GoodStudent
