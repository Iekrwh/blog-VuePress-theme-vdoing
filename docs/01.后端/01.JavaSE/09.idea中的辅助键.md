# idea中的辅助键

快速生成语句

1. 快速生成mian()方法:psvm 回车
2. 快速生成输出语句:sout  回车

内容辅助键

Ctrl+Alt+space (内容提示/代码补全)

快捷键

单行注释 ctrl+/

多行注释 ctrl+shift+/

格式化 ctrl+alt+l

快速补全类型和变量名 ctrl+alt+v



## 类中创建方法

alt + insert  可快速创建构造方法、set和get方法
