# 提取4位数的各位数

> 1234//1000=1.234整除=1  提取第一位             #整除1
> 1234 //100=12.34=12%10=2 提取第二位				#余2
> 1234 //10=123.4=123%10=120 提取第三位     #余3
> 1234%10 =1234-1230= 余4   提取第四位         #余4
