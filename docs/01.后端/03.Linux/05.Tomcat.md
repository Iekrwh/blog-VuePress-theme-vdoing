# Tomcat

上传linux

```sh
tar -zxvf apache-tomcat-9.0.52.tar.gz

cd apache-tomcat-9.0.52/bin

./startup.sh
```



## yum安装

```sh
yum -y install tomcat

cd /usr/share/tomcat

systemctl status tomcat

systemctl start tomcat

# 安装web插件包
yum install tomcat-webapps tomcat-admin-webapps
```
