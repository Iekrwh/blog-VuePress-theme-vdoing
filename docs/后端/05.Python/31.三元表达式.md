# 三元表达式

> ```python
> #3. 三元表达式(if...else语句的简写),将if...else转变成一行代码
> num1 = 12
> num2 = 42
> # if num1 < num2:
> #     x = num2
> # else:
> #     x = num1
> x = num1 if num1 > num2 else num2   # a if 1 > 2 else b,意思是如果1>2成立 则输出a否则输出b
> print(x,'这数大')
> ```
