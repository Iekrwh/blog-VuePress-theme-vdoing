# 数据输入

1. 导包

   import java.util.Scanner;

   导包的动作必须出现在类定义的上边

2. 创建对象

   Scanner sc =new Scanner(System.in);

   只有sc是变量名，可以变，其他的都不允许变

3. 接受数据

   int i = sc.nextInt();

   只有i是变量名，可以变，其他的也不允许变
