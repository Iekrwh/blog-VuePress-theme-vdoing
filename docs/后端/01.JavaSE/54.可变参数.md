# 可变参数

可变参数:就是形参的个数是可以变换的



```java
public static void main(String[] args) {
    int sum = getSum(1,5,5,3135,156,1456,1);
    System.out.println(sum);
}
public static int getSum(int... arr){
    int sum = 0;
    for (int i : arr) {
        sum +=i;
    }
    return sum;
}
```

注意:

- 底层其实就是一个数组存储

- 可变参数必须写在最后
