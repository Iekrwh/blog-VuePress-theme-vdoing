# 抽象类与接口的区别

成员区别:

抽象类   变量,常量;有构造方法;有抽象方法;也有非抽象方法

接口 常量;抽象方法



关系区别:

类与类   继承,单继承

类与接口 实现,可单实现,也可多实现

接口与接口 继承,单继承,多继承



设计理念区别:

抽象类  对类抽象,包括属性、行为

接口 对行为抽象，主要是行为



抽象类是对事物的抽象,而接口是对行为的抽象
